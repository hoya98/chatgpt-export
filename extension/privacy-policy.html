Privacy Policy

Effective Date: February 10, 2026

ChatGPT Export is a browser extension that helps you export your own ChatGPT conversation data.

Data Collection

This extension does not collect, store, transmit, or share any user data. Period.

How It Works

The extension runs entirely in your browser. It:

  • Reads your existing ChatGPT session token (already in your browser cookies) to authenticate API requests
  • Calls ChatGPT's internal API endpoints to retrieve your own conversation data
  • Packages the data into a JSON file and downloads it directly to your computer

All processing happens locally in your browser. No data is sent to any external server, analytics service, or third party.

Permissions

  • activeTab: Required to interact with the chatgpt.com page when you click the export button
  • cookies: Required to detect your workspace account ID for Team/Business workspace exports
  • host_permissions (chatgpt.com): Required to run the export script on the ChatGPT website

Third-Party Services

This extension communicates only with chatgpt.com (OpenAI's servers) using your existing authenticated session. It does not communicate with any other servers or services.

Data Storage

The extension does not store any data in browser storage, local storage, cookies, or anywhere else. The exported file is saved directly to your Downloads folder.

Open Source

This extension is fully open source. You can review the complete source code at github.com/hoya98/chatgpt-export.
